Getting everything set up

There are a small number of tools you need in order to create and deploy Streamlit apps.

Software

Python

Streamlit is a Python package, and requires Python 3.6+. To verify you have an appropriate version of Python, run:

python --version

from a command line. If you see a Python 2 version then try

python3 --version

instead.

Tip
If you are installing Python on Windows, make sure you tick the “add to path” option when presented.

Any text editor

We will be writing some Python code. Any text editor will will be absolutely fine, but a couple of recommendations are:

  • Visual Studio Code
  • PyCharm Community

Git

We will be working with a Git repository hosted on GitHub. You should make sure you have installed Git. You can either use Git from the command line, or use a graphical application such as

  • GitHub Desktop
  • GitKraken

Services

GitHub

We will be using GitHub to host a Git repository for this workshop. Please make sure you have a GitHub account and that you are logged in.

Heroku

Heroku is a cloud platform for hosting apps. We will be using Heroku to host our Streamlit apps so they are live for everyone on the internet to play with. Please make sure you have a Heroku account and that you are logged in.
